  <Step title="Navigate through the sections">
    Use the horizontal menu to explore:
    - **Inicio** (Home): Project overview and introduction
    - **Análisis Exploratorio**: Visualize global trends, cases, deaths, tests, and vaccinations
    - **Clustering**: Explore country groupings by socioeconomic factors and temporal evolution

    - Filter data by continent, country, or time period
    - View correlation heatmaps between socioeconomic variables
    - Explore K-Shape clustering results for daily cases patterns
  </Step>
</Steps>

## Option 2: Run Locally

For deeper analysis and access to Jupyter notebooks, run the project on your local machine.

<Steps>
  <Step title="Clone the repository">
    ```bash
    git clone https://github.com/yourusername/covid19-data-analytics.git
    cd covid19-data-analytics
    ```
  </Step>
  
  <Step title="Set up environment">
    Create and activate a virtual environment, then install dependencies:
    
    ```bash
    python -m venv venv
    source venv/bin/activate  # On Windows: venv\Scripts\activate
    pip install -r requirements.txt
    ```
    
    <Note>
      This project requires Python 3.8+ and installs 130+ packages including Streamlit, pandas, scikit-learn, and PyCaret.
    </Note>
  </Step>
  
  <Step title="Extract datasets">
    The datasets are compressed to save space. Extract them before running:
    
    ```bash
    cd covid_app/datasets/inputs
    unzip owid-covid-data.zip
    unzip countries-lat-lon.zip
    unzip HDR25_Composite_indices_complete_time_series.zip
    ```
  </Step>
  
  <Step title="Launch the Streamlit app">
    ```bash
    cd covid_app
    streamlit run app.py
    ```
    
    The app will open in your browser at `http://localhost:8501`
  </Step>
</Steps>

## Understanding the App Structure

The Streamlit application consists of three main modules:

```python
# Main app entry point (app.py)
from streamlit_option_menu import option_menu

selected = option_menu(
    menu_title=None, 
    options=["Inicio", "Análisis Exploratorio", "Clustering"],
    icons=["house", "bar-chart", "diagram-3"],
    orientation="horizontal"
)
```

Each section is loaded dynamically from its own module:
- `inicio.py` - Home page with project overview
- `eda.py` - Exploratory data analysis visualizations
- `clustering.py` - Clustering results and interactive maps
