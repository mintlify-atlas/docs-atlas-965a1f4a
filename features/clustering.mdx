# Clustering Analysis

The clustering module applies unsupervised machine learning techniques to identify countries with similar COVID-19 dynamics. This grouping reveals common patterns and enables comparative analysis across countries with shared characteristics.

## Clustering Approaches

The analysis implements three distinct clustering strategies:

### 1. Socioeconomic Clustering (K-Means)

**Method**: K-Means with k=3 clusters  
**Features**: GDP per capita, median age, life expectancy, population density, stringency index

<Accordion title="K-Means Implementation">
  ```python
  # From clustering.ipynb
  def clusterizar(
      df, features, n_clusters=3, 
      scaler=StandardScaler(),
      method="kmeans"
  ):
      # Scale features
      X_scaled = scaler.fit_transform(features)
      
      # Apply K-Means clustering
      model = KMeans(
          n_clusters=n_clusters, 
          n_init=20, 
          random_state=42
      )
      labels = model.fit_predict(X_scaled)
      
      # Add cluster assignments
      df_clustered = features.copy()
      df_clustered['cluster'] = (labels + 1).astype(str)
  ```
  
  The algorithm groups countries based on static socioeconomic characteristics.
</Accordion>

## Dimensionality Reduction (PCA)

Principal Component Analysis reduces feature dimensionality while preserving variance:

```python
# From clustering.ipynb:244-303
def aplica_acp(df, varexplicada=0.85, scaler=StandardScaler()):
    # Scale features
    scaler.set_output(transform="pandas")
    ts_features_scaled = scaler.fit_transform(df)
    
    # PCA to retain 85% variance
    pca = PCA(n_components=varexplicada)
    pca_projections = pca.fit_transform(ts_features_scaled)
    
    print(f"Number of components: {pca.n_components_}")
    print(f"Variance explained: {pca.explained_variance_ratio_.sum()*100:.2f}%")
    
    return pca_projections
```

<Tip>
  PCA typically reduces 700+ features to 10-20 principal components while retaining 85% of variance, improving clustering efficiency.
</Tip>

## Optimal Cluster Selection

Silhouette score determines the optimal number of clusters:

```python
# From clustering.ipynb:306-392
def seleccionar_mejor_k_clusters(
    data,
    rango_clusters=range(2, 11),
    method="kmeans"
):
    silhouette_scores = []
    
    for n_clusters in rango_clusters:
        if method == "kmeans":
            model = KMeans(n_clusters=n_clusters, random_state=42)
        elif method == "kmedoids":
            model = KMedoids(n_clusters=n_clusters, random_state=42)
        
        labels = model.fit_predict(data)
        score = silhouette_score(data, labels)
        silhouette_scores.append(score)
    
    best_n = silhouette_scores.index(max(silhouette_scores)) + 2
    return best_n, max(silhouette_scores)
```

<Note>
  Silhouette scores range from -1 to 1. Higher values indicate better-defined clusters. Scores above 0.5 suggest strong clustering structure.
</Note>
