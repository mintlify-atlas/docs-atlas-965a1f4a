# Streamlit Interactive Application

The project includes a fully functional **Streamlit web application** that provides an interactive interface to explore the COVID-19 data analysis results. Access the live application or run it locally for customized analysis.

<Card title="Live Application" icon="external-link" href="https://covid19-data-analytics-owid-xasv8wrmsgkmtautyfywth.streamlit.app/">
  **Access the deployed app instantly** - No installation required!
</Card>

**Available Filters**:
- **Continent**: Focus on Africa, Asia, Europe, North America, Oceania, or South America
- **Country**: Drill down to any of 191 countries
- **Indicator**: Choose Cases, Deaths, Tests, or Vaccinations
- **KPI Mode**: Toggle between absolute totals and per-capita normalized values

#### Key Performance Indicators (KPIs)

```python
# From covid_app/eda.py:93-122
def mostrar_kpis(df, mode="Totales"):
    latest = df[df['date'] == df['date'].max()]
    if mode == "Totales":
        kpi_data = [
            ("Casos totales", f"{latest['total_cases'].sum():,.0f}"),
            ("Muertes totales", f"{latest['total_deaths'].sum():,.0f}"),
            ("% Vacunados", f"{latest['vaccination_ratio'].mean():.1f}%"),
            ("Total Pruebas", f"{latest['total_tests'].sum():,.0f}")
        ]
    else:  # Normalizados
        kpi_data = [
            ("Casos / millón", f"{latest['total_cases_per_million'].sum():,.0f}"),
            ("Muertes / millón", f"{latest['total_deaths_per_million'].sum():,.0f}"),
            ("% Vacunados", f"{latest['vaccination_ratio'].mean():.1f}%"),
            ("Pruebas / mil", f"{latest['total_tests_per_thousand'].sum():,.0f}")
        ]
```

KPIs dynamically update based on selected filters.

## Section 3: Clustering

### Interactive Cluster Exploration

The clustering section allows users to:

1. **Select clustering criterion**
   - Socioeconomic factors (K-Means, k=3)
   - Daily cases evolution (K-Shape, k=4)
   - Daily deaths evolution (K-Shape, k=3)

2. **Choose cluster to visualize**  
   Radio buttons to select cluster 1, 2, 3, or 4

3. **Select specific country**  
   Dropdown to visualize individual country within the cluster

```python
# From covid_app/clustering.py:255-280
col1, col2, col3 = st.columns(3)
```

## Running Locally

### Prerequisites

```bash
pip install streamlit pandas numpy plotly streamlit-option-menu folium streamlit-folium
```

### Launch the App

<Steps>
  <Step title="Clone Repository">
    ```bash
    git clone https://github.com/your-repo/covid19-data-analytics.git
    cd covid19-data-analytics
    ```
  </Step>
  
  <Step title="Decompress Datasets">
    ```bash
    cd covid_app/datasets/outputs
    unzip complete-datasets.zip
    unzip filtered-datasets.zip
    ```
  </Step>
  
  <Step title="Update File Paths">
    Edit `covid_app/eda.py` and `covid_app/clustering.py` to use local paths if needed.
  </Step>
  
  <Step title="Run Streamlit">
    ```bash
    cd covid_app
    streamlit run app.py
    ```
  </Step>
</Steps>

<Note>
  The app will open in your browser at `http://localhost:8501`
</Note>

## App Architecture

```
covid_app/
├── app.py              # Main entry point, navigation
├── inicio.py           # Home page, dataset loading
├── eda.py              # Exploratory analysis page
├── clustering.py       # Clustering analysis page
├── constantes.py       # Constants (continent colors, translations)
├── datasets/
│   ├── inputs/         # Raw OWID data (compressed)
│   └── outputs/        # Processed datasets
│       ├── complete-datasets/
│       ├── filtered-datasets/
│       └── cluster-countries.csv
└── requirements.txt    # Python dependencies
```

### Session State Management

```python
# From covid_app/eda.py:778-809
if "datasets" not in st.session_state:
    st.warning("Debe situarse en la página de Inicio para cargar los datos")
    st.stop()

df, df_weekly, df_monthly, df_quarterly, df_yearly = st.session_state.datasets

if "last_df" not in st.session_state:
    st.session_state.last_df = df.sort_values('date').groupby('country', observed=True).tail(1)

if "df_map" not in st.session_state:
    st.session_state.df_map = prepare_map_data(st.session_state.last_df)
```

<Tip>
  **Session state** persists data across page navigation, avoiding redundant loading and improving performance.
</Tip>
