# Exploratory Data Analysis

The Exploratory Data Analysis (EDA) module provides a comprehensive analysis of the global COVID-19 pandemic evolution using data from [Our World in Data](https://ourworldindata.org/coronavirus). This analysis helps understand the factors associated with virus incidence and mortality across different countries and continents.

## Key Variables Analyzed

### Pandemic Metrics
- **Cases**: Daily and cumulative cases (total and per million)
- **Deaths**: Daily and cumulative deaths (total and per million) 
- **Tests**: Testing capacity and positivity rates (per thousand)
- **Vaccinations**: Total vaccinations and vaccination ratio

### Socioeconomic Factors
- **GDP per capita**: Economic development indicator
- **Median age**: Population age distribution
- **Stringency index**: Government response measures (0-100 scale)
- **Life expectancy**: Health system quality indicator

## Data Aggregation Levels

The analysis supports multiple temporal aggregations:

- **Daily**: 278,991 records across 191 countries (2020-01-01 to 2023-12-31)
- **Weekly**: 40,110 records for trend analysis
- **Monthly**: 9,168 records for medium-term patterns
- **Quarterly**: 3,056 records for seasonal analysis
- **Yearly**: 764 records for long-term trends

  ```python
  # From covid_app/eda.py:156-234
  def mostrar_graficaComparativa(last_df):     
      continent_avg = last_df.groupby('continent', observed=True)[[
          'total_cases_per_million', 
          'total_deaths_per_million', 
          'total_tests_per_thousand',
          'vaccination_ratio'
      ]].mean().reset_index()

      # Normalize each column to max for visual comparison
      cols = ['total_cases_per_million', 'total_deaths_per_million',
              'total_tests_per_thousand', 'vaccination_ratio']
      continent_avg_norm = continent_avg.copy()
      for col in cols:
          continent_avg_norm[col] = continent_avg_norm[col] / continent_avg_norm[col].max()
  ```
  
  Normalized values allow direct comparison across continents with different scales.
</Accordion>

## Key Findings

Based on the EDA visualizations:

1. **Asia** has the largest population but not proportionally higher case rates
2. **Economic development** (GDP per capita) strongly correlates with testing capacity
3. **Vaccination campaigns** show clear inverse relationship with mortality
4. **Government restrictions** show delayed effects (15-60 days) on transmission
5. **Median age** significantly influences mortality rates
