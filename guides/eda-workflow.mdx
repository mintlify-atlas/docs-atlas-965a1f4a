# Exploratory Data Analysis Workflow

This guide provides a step-by-step walkthrough of the exploratory data analysis (EDA) process used in the COVID-19 Data Analytics project. The EDA workflow helps understand data patterns, distributions, and relationships before modeling.

## Overview

The EDA process analyzes the preprocessed COVID-19 dataset to understand:
- How cases and deaths have progressed over time
- Factors influencing pandemic evolution (socioeconomic characteristics, government measures, vaccination campaigns)
- Distribution patterns across countries and continents

<Steps>

### Load Libraries and Data

First, import necessary libraries for data manipulation, visualization, and feature extraction:


### Analyze Distribution by Continent

Visualize countries and population distribution across continents:

```python
continent_counts = df.groupby("continent", observed=False)['country'].nunique().reset_index()
continent_counts.columns = ['continent', 'num_countries']
continent_counts['population'] = df.groupby("continent", observed=False)['population'].sum().values

fig = go.Figure()

# Bar chart: number of countries
fig.add_trace(go.Bar(
    x=continent_counts["continent"],
    y=continent_counts["num_countries"],
    name="Number of Countries",
    marker=dict(color="steelblue"),
    opacity=0.85,
    yaxis="y1"
))

# Line chart: total population
fig.add_trace(go.Scatter(
    x=continent_counts["continent"],
    y=continent_counts["population"],
    name="Total Population",
    mode="lines+markers",
    marker=dict(color="#ff7f0e", size=10),
    line=dict(color="#ff7f0e", width=3),
    yaxis="y2"
))

fig.update_layout(
    title="Countries and Population by Continent",
    xaxis=dict(title=None),
    yaxis=dict(title="Number of Countries"),
    yaxis2=dict(
        title="Total Population",
        overlaying="y",
        side="right"
    )
)

fig.show()
```

<Tip>
Africa has the most countries (53), but Asia has the largest total population, tripling Africa's and quadrupling Europe's population.
</Tip>

### Time Series Visualization

Visualize trends over time for specific countries:

```python
def plot_country_timeseries(df, country, metric):
    """
    Plot time series for a specific country and metric
    """
    df_country = df[df['country'] == country].copy()
    
    fig = go.Figure()
    fig.add_trace(go.Scatter(
        x=df_country['date'],
        y=df_country[metric],
        mode='lines',
        name=metric
    ))
    
    fig.update_layout(
        title=f"{metric.replace('_', ' ').title()} - {country}",
        xaxis_title="Date",
        yaxis_title=metric.replace('_', ' ').title(),
        template="plotly_white"
    )
    
    fig.show()

# Example: Plot new cases for Spain
plot_country_timeseries(df, 'Spain', 'new_cases_smoothed_per_million')
```

### Feature Extraction with TSFresh

Extract time series features for advanced analysis:

```python
from tsfresh import extract_features
from tsfresh.utilities.dataframe_functions import impute

# Prepare data for TSFresh
ts_data = df[['country', 'date', 'new_deaths_smoothed_per_million']].copy()
ts_data['id'] = ts_data['country']
ts_data['time'] = ts_data.groupby('country').cumcount()

# Extract features
extracted_features = extract_features(
    ts_data,
    column_id='id',
    column_sort='time',
    column_value='new_deaths_smoothed_per_million',
    impute_function=impute,
    show_warnings=False
)

print(f"Extracted {extracted_features.shape[1]} features")
```

<Warning>
Feature extraction can be computationally intensive for large datasets. Consider using a subset of data for initial exploration.
</Warning>
