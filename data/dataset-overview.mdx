This project analyzes the global evolution of the COVID-19 pandemic using open data from [Our World in Data](https://ourworldindata.org/coronavirus). The dataset provides comprehensive COVID-19 statistics combined with socioeconomic indicators for countries worldwide.

## Data Source

The dataset is sourced from the Our World in Data COVID-19 repository, which aggregates data from multiple authoritative sources including:
- Johns Hopkins University
- World Health Organization (WHO)
- National government health departments
- Testing and vaccination data from official sources

The dataset is regularly updated and can be accessed at:
```
https://catalog.ourworldindata.org/garden/covid/latest/compact/compact.csv
```

<Note>
Datasets included in this repository are **compressed** (`.zip` format) due to GitHub storage limitations. Before executing notebooks or scripts (except the Streamlit app), you should **decompress the files** in the corresponding folder while maintaining the directory structure.
</Note>

## Time Period Covered

The analysis focuses on data from **January 1, 2020** to **December 31, 2023**, covering the critical years of the pandemic including:
- Initial outbreak and first wave (2020)
- Multiple variant waves (2021-2022)
- Vaccination rollout period (2021-2023)
- Post-pandemic transition (2023)

## Geographic Coverage

The dataset includes **195 sovereign countries** across all continents:
- **Asia**: 49 countries
- **Africa**: 54 countries  
- **Europe**: 44 countries
- **Americas**: 35 countries (North and South America)
- **Oceania**: 13 countries

Non-sovereign territories and regional aggregates are excluded from the analysis to ensure consistency.

## Data Structure

The dataset is structured as a **panel/longitudinal dataset** with:
- **Daily observations** for each country
- **Time series format**: Each row represents one country on one date
- **Primary keys**: `country` and `date`
- **Format**: CSV (Comma-Separated Values)

## Data Quality Considerations

<Note>
The raw dataset contains significant missing values, particularly for:
- **Hospital and ICU admission data**: Limited reporting in many countries
- **Testing data**: Inconsistent reporting, especially in early 2020
- **Vaccination data**: Only available from late 2020 onwards
- **Socioeconomic indicators**: Some countries have incomplete data

These issues are addressed in the [data cleaning](/data/data-cleaning) and [preprocessing](/data/preprocessing) pipelines.
</Note>

## Data Updates

The Our World in Data repository is updated regularly, but this analysis uses a snapshot covering 2020-2023. For the most current data, visit the [Our World in Data COVID-19 page](https://ourworldindata.org/coronavirus).
