## Overview

The `eda` module (located in `covid_app/eda.py`) provides a comprehensive interactive Streamlit application for exploring COVID-19 data. It features KPI dashboards, temporal visualizations, geospatial maps, correlation analysis, and policy impact assessment.

## Import

```python
from covid_app import eda
```

---

## Main Function

### main()

Launches the interactive exploratory data analysis Streamlit application.

```python
eda.main()
```

**Parameters:** None (reads from `st.session_state`)

**Returns:** None (renders Streamlit UI)

<Note>
**Application Features:**
- Interactive filters (continent, country, indicator, KPI mode)
- Key performance indicators (KPIs) with totals/normalized views
- Top/bottom country rankings
- Temporal evolution charts (cases, deaths, vaccinations, tests)
- Continental comparison analysis
- Scatter plots with socioeconomic correlations
- Stringency index analysis with LOWESS curves
- Interactive Folium map with country markers
</Note>

#### Example

```python
import streamlit as st
from covid_app import eda

# Run as Streamlit app
if __name__ == "__main__":
    eda.main()
```

## Filter Functions

### reset_filtros()

Resets all filters to default values in `st.session_state`.

```python
eda.reset_filtros()
```

**Returns:** None (modifies session state)

**Default Values:**
- Continent: "Todos" (All)
- Country: "Todos" (All)
- Indicator: "Casos" (Cases)
- KPI Mode: "Totales" (Totals)

---

### aplicar_filtros()

Applies continent and country filters to the DataFrame.

```python
eda.aplicar_filtros(df, seleccion_continente, mapping_continente, seleccion_country, mapping_pais=None)
```

<ParamField path="df" type="pd.DataFrame" required>
  DataFrame to filter
</ParamField>

<ParamField path="seleccion_continente" type="str" required>
  Selected continent name (in Spanish) or "Todos"
</ParamField>

<ParamField path="mapping_continente" type="dict" required>
  Dictionary mapping Spanish continent names to English names
</ParamField>

<ParamField path="seleccion_country" type="str" required>
  Selected country name (in Spanish) or "Todos"
</ParamField>

<ParamField path="mapping_pais" type="dict | None" default="None">
  Dictionary mapping Spanish country names to English names (required if country selected)
</ParamField>

**Returns:** `pd.DataFrame` - Filtered DataFrame

#### Example

```python
from constantes import CONTINENTS_SPANISH, PAISES_SPANISH

# Filter by continent
df_europe = eda.aplicar_filtros(df, "Europa", CONTINENTS_SPANISH, "Todos")

# Filter by country
df_spain = eda.aplicar_filtros(df, "Europa", CONTINENTS_SPANISH, "España", PAISES_SPANISH)
```

</ParamField>

</ParamField>

<ParamField path="title_bottom" type="str" default="''">

</ParamField>

<ParamField path="xlabel" type="str" default="''">

</ParamField>

</ParamField>

<ParamField path="indicador_nombre" type="str" default="''">

### scatter_plots()

Creates a three-panel scatter plot exploring socioeconomic correlations.

```python
eda.scatter_plots(last_df, continent_colors)
```

<ParamField path="last_df" type="pd.DataFrame" required>
  DataFrame with latest values per country
</ParamField>

<ParamField path="continent_colors" type="dict" required>
  Dictionary mapping continent names to colors
</ParamField>

**Returns:** None (renders Plotly subplot with 3 columns)

**Panels:**

1. **Tests vs Cases (GDP influence)**
   - X: Cases per million (log scale)
   - Y: Tests per thousand (log scale)
   - Size: GDP per capita

2. **GDP vs Vaccination Coverage**
   - X: GDP per capita
   - Y: Vaccination percentage
   - Fixed marker size

3. **Cases vs Deaths (Age influence)**
   - X: Cases per million (log scale)
   - Y: Deaths per million (log scale)
   - Size: Median age (squared for visibility)

#### Example

```python
from constantes import CONTINENT_COLORS

latest_df = df.sort_values('date').groupby('country').tail(1)
eda.scatter_plots(latest_df, CONTINENT_COLORS)
```

### curvas_lowess()

Creates interactive LOWESS regression curves showing the impact of policy stringency on cases and deaths with adjustable time lags.

```python
eda.curvas_lowess(df, continente, pais="Todos")
```

<ParamField path="df" type="pd.DataFrame" required>
  Time series DataFrame
</ParamField>

<ParamField path="continente" type="str" required>
  Selected continent name
</ParamField>

<ParamField path="pais" type="str" default="'Todos'">
  Selected country name
</ParamField>

**Returns:** `plotly.graph_objects.Figure` with interactive lag selector

**Features:**
- **Lag Options**: 0, 15, 30, 40, 60, 80 days
- **Method**: LOWESS (Locally Weighted Scatterplot Smoothing) with `frac=0.3`
- **Subplots**: Daily cases (top), Daily deaths (bottom)
- **Interactive**: Dropdown menu to change lag period

<Note>
LOWESS curves reveal non-linear relationships between stringency and outcomes. Time lags account for policy implementation delays and incubation periods.
</Note>

#### Example

```python
fig = eda.curvas_lowess(df_filtered, "Europa", "España")
fig.update_layout(height=600)
st.plotly_chart(fig, use_container_width=True)
```

## Map Functions

### create_country_map()

Creates an interactive Folium map with clustered country markers.

```python
eda.create_country_map(df_map)
```

<ParamField path="df_map" type="pd.DataFrame" required>
  DataFrame with columns: `country`, `lat`, `lon`, `total_cases`, `total_deaths`, `total_vaccinations`, `radius`
</ParamField>

**Returns:** `folium.Map` - Folium map object

**Map Features:**
- **Base Layer**: CartoDB Positron (clean, light theme)
- **Marker Clustering**: Groups nearby countries at low zoom levels
- **Color Scale**: Blue gradient based on total cases
- **Popup**: Shows country name, cases, deaths, vaccinations
- **Circle Size**: Proportional to log(cases)

#### Example

```python
from streamlit_folium import st_folium

# Prepare map data
latest_df = df.sort_values('date').groupby('country').tail(1)
df_map = latest_df[['country','lat','lon','total_cases','total_deaths','total_vaccinations']].dropna()
df_map['radius'] = 5 + (np.log10(df_map['total_cases'] + 1) - df_map['total_cases'].min()) * 2

# Create and display map
m = eda.create_country_map(df_map)
st_folium(m, width=None, height=600)
```
