## Overview

This project provides a complete data analytics platform for understanding the COVID-19 pandemic through multiple analytical lenses. Built on top of the comprehensive [Our World in Data](https://ourworldindata.org/coronavirus) COVID-19 dataset, it combines exploratory data analysis, machine learning clustering, and time-series forecasting into an interactive Streamlit application.

## Key Features

<AccordionGroup>
  <Accordion title="Interactive Streamlit Dashboard" icon="chart-line">
    Explore COVID-19 data through an intuitive web interface with multiple analysis views:
    - **Inicio**: Project overview and navigation
    - **Análisis Exploratorio**: Global and continental trend visualization
    - **Clustering**: Pattern discovery across countries
  </Accordion>

  <Accordion title="K-Shape Clustering for Temporal Patterns" icon="diagram-project">
    Advanced clustering algorithms identify countries with similar COVID-19 trajectory patterns:
    - Socioeconomic factor-based grouping
    - Temporal evolution clustering using K-Shape algorithm
    - Daily cases and deaths pattern analysis
  </Accordion>

  <Accordion title="SARIMAX Time-Series Forecasting" icon="chart-area">
    Predictive modeling for COVID-19 deaths in Spain:
    - One-day forecast horizon
    - Vaccination rate as exogenous variable
    - **RMSE: 0.078** | **MAE: 0.0197**
  </Accordion>

  <Accordion title="Comprehensive Exploratory Analysis" icon="magnifying-glass-chart">
    Deep dive into global COVID-19 trends:
    - Cases, deaths, tests, and vaccination tracking
    - Correlation analysis with socioeconomic indicators
    - GDP per capita, median age, and stringency index studies
  </Accordion>
</AccordionGroup>

## Project Structure

```
covid19-data-analytics-owid/
├── covid_app/              # Streamlit web application
│   ├── app.py             # Main application entry point
│   ├── inicio.py          # Home page
│   ├── eda.py             # Exploratory analysis page
│   └── clustering.py      # Clustering analysis page
├── cleaning.ipynb         # Data cleaning workflow
├── eda.ipynb              # Exploratory data analysis
├── clustering.ipynb       # Clustering analysis
├── models.ipynb           # Predictive modeling
├── aux_functions.py       # Utility functions
└── datasets/              # Compressed data files
```

<Tip>
  Datasets in the repository are compressed. Extract them before running notebooks locally, and update file paths in the code accordingly.
